# 🌿 ANOHA – Waste Classifier

ANOHA is an AI-powered waste classification app that identifies waste items from images and provides proper disposal, recycling, and waste management guidance.

## Features
- 📷 Upload or take a photo of any waste item
- 🤖 AI instantly identifies the item using Claude Vision
- ♻️ Get the correct disposal category and instructions
- 📚 Learn about waste management techniques

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Vercel Serverless Functions
- AI: Claude API (Anthropic)
